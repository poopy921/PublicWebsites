What is Brainfuck?

Brainfuck is an esoteric programming language that only consists of 8 characters! Everything else is treated like a comment. It is Turing complete, meaning that you can theoretically make anything out of it.

How does it work?

Imagine that you're operating an infinitely long array of cells, with a pointer indicating where operations take place. These characters revolve around adding and subtracting from the cells, getting input and output, looping, and moving around the pointer.
